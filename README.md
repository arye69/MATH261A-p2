# MATH261A Final Project
Final project repository for MATH 261A.

## Data 

Data used in this project is obtained from [SCORE Sports Data Repository](https://data.scorenetwork.org/basketball/nba_draft_1990-2021.html). According to SCORE, the data were pulled from [Kaggle](https://www.kaggle.com/datasets/benwieland/nba-draft-data), who originally sourced their data from [National Collegiate Athletic Association](https://www.ncaa.org/sports/2017/9/5/ncaa-research-interactive-databases.aspx) (NCAA). The license for this data (on the Kaggle level) is CC0: Public Domain.
